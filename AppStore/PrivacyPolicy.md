# MathFeedback Privacy Policy

Last updated: June 13, 2026

MathFeedback is a local-first app for recording after-class mathematics feedback.

## Data Collection

MathFeedback does not sell or share personal data. Student names, class names, notes, feedback records, ratings, homework notes, and exported reports are stored locally on the user's device by default.

## Data Storage

The app stores data using Apple's SwiftData framework on the device. The developer does not operate a cloud backend for storing app data.

Users can export a JSON backup from the app and choose where to save or share it. Once exported, the user controls that file and is responsible for where it is stored.

## Third-Party Services

MathFeedback does not use third-party analytics, advertising, tracking SDKs, or a developer-operated cloud backend.

If the user enters a DeepSeek API key in Settings and taps the AI generation button, the app sends the relevant feedback inputs to the DeepSeek API to generate a teacher comment. These inputs may include the lesson topic, learning content, overall rating, skill scores, homework completion, class participation, strengths, and improvement points. The DeepSeek API key is stored only in the device Keychain.

## Children's Data

The app may be used by teachers or tutors to record student-related notes. This information remains on the user's device unless the user exports it, shares it, or uses the optional DeepSeek-powered AI generation feature. Users should protect their device, backups, exported reports, and DeepSeek API key appropriately.
